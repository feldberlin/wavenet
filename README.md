# Wavenet

An unconditioned Wavenet implementation with fast generation.


## Installation

Requires python 3.7.2 or greater, will install into a virtualenv.

```
bin/install
```

## Training

Use a training notebook like this:

```
bin/train notebooks/train.maestro.ipynb
```
